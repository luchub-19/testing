# Task Planner Skill

Use this skill when the task requires multi-step planning.

## Strategy
1. Break the task into clear sub-goals
2. Identify which tools are needed for each sub-goal
3. Execute in order, verify each result before proceeding
4. If a step fails, try an alternative approach before giving up

## Tips
- Use `memory` tool to store intermediate results
- Use `calculator` for any arithmetic — never compute in your head
- Use `exec` for file system operations when `file` tool is insufficient
